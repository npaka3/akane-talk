## ★お兄ちゃん! いっしょに おはなし しよ♪

あかねちゃん は はるかちゃん の妹の女の子。
ふたりともきんじょに住んでいる“お兄ちゃん”が大好き。
最近は、 ヲタク なことに興味津
“お兄ちゃん”にいろいろ教えてもらっています。
そんなある日、 あかねちゃん は“お兄ちゃん”と
いっしょに おはなし することになりました。
季節は夏風味。
夏休み中風味なので、時間はたっぷりあります。
さてさて、どんな楽しい おはなし になるのでしょう?

## ★「あかねと〜く」ってなぁに?
